## Simulación de Cuerpos Celestes

# Descripción
Este proyecto simula el movimiento de cuerpos celestes como planetas en órbitas elípticas, basándose en las leyes de Kepler. La simulación está escrita en Python y utiliza Programación Orientada a Objetos (POO), decoradores personalizados, y el método numérico Runge-Kutta para resolver ecuaciones diferenciales. Se incluye además una animación para visualizar el movimiento orbital en coordenadas cartesianas.
